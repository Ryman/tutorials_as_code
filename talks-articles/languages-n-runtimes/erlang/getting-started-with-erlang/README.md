### Chapter 1.1

It is an introduction, which only consists simplest constructs to comprehend. It skips on

* References
* Local error handling
* Single direction links
* Handling of binary data
* List comprehensions
* Interoperability
* OTP
* Hash tables for Erlang terms (ETS)
* Live logic changes
